# milk_RNA_profile
This repository contains the scripts and associated documents related to the publication entitled "Gene expression profiling of bovine raw milk is linked to the inflammatory status of the udder" by Gitton et al.

Four scripts are provided:
  - a script for the analysis of sorted cells and clustering of genes
  - a script for the analysis of milk cells
  - a script for the analysis of flow cytometry data
  - a script for comaprison of dendrograms

Data to run these different scripts are provided in the data folder.
